# RestAPI

練習Restful設計風格，完成一套CRUD，使用Postman測試API

查詢所有會員<br>
GET<br>
http://localhost:8080/users<br>
查詢單一會員<br>
GET<br>
http://localhost:8080/user/1<br>
新增會員<br>
PUT<br>
http://localhost:8080/save/user<br>
更新會員<br>
PATCH<br>
http://localhost:8080/update/user/1<br>
刪除會員<br>
DELETE<br>
http://localhost:8080/delete/user/1<br>
